## SkillOps

Visibility: `private`

### Use in SkillOps Desktop

1. Open SkillOps.
2. Click **Add Skill project**.
3. Enter `github.com/feitianchengzi/arckit/tree/main/definition` as the GitHub source.
4. Choose a profile and add an application target.

### Profiles

- `default`: all skills
- `spec`: arckit-spec

### CLI

```bash
skillshare install github.com/feitianchengzi/arckit/tree/main/definition --track --all && skillshare sync
npx skills add github.com/feitianchengzi/arckit/tree/main/definition
```

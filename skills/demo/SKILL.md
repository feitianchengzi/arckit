# Demo

Turn short input into a crisp three-bullet summary.

## Workflow

1. Identify the core topic, important details, and implied action items.
2. Return exactly three bullets unless the user requests another format.
3. Keep each bullet under 20 words.
4. Use plain language and preserve concrete names, dates, numbers, and decisions.
5. If the input is ambiguous, state the uncertainty inside the relevant bullet instead of asking a follow-up.

## Output Shape

```markdown
- Main point...
- Important detail...
- Next action or open question...
```

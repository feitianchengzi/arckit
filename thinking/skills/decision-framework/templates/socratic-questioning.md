## 一、澄清概念与事实

> 目标：统一语言。避免讨论了半小时发现各说各话。

### 1.1 核心概念定义

| 关键词 | 我以为的意思 | 可能的歧义 | 统一定义 |
|--------|------------|-----------|---------|
| | | | |
| | | | |
| | | | |

引导追问：
- 你说的"XX"具体指什么？能举个具体例子吗？
- 这个词在这里是狭义还是广义？
- 有没有可能别人听到这个词理解的是另一回事？

### 1.2 事实与判断分离

| 陈述 | 事实 🔲 | 判断 🔲 | 来源 |
|------|---------|---------|------|
| | | | |
| | | | |
| | | | |

引导追问：
- 这句话里，哪些是可以验证的事实？哪些是你的推断/感受？
- 这个数据的来源是什么？一手还是二手？
- 有没有被包装成事实的价值判断？

## 二、探查假设与前提

> 目标：挖出命题背后默认成立、但未经检验的信念。**这是最关键的层次。**

### 2.1 默认假设清单

| 编号 | 隐含假设 | 为什么可以这样默认？ | 如果不成立会怎样？ |
|------|---------|-------------------|------------------|
| A1 | | | |
| A2 | | | |
| A3 | | | |
| A4 | | | |

引导追问：
- 这个论点似乎默认了……，请问为什么可以这样默认？
- 如果这个假设不成立，结论还站得住吗？
- 这个前提总是正确的吗？有没有例外情况？
- 有没有我们"太熟悉以至于忘记质疑"的前提？

### 2.2 假设脆弱度评估

| 假设 | 确定程度 | 验证方式 | 验证成本 |
|------|---------|---------|---------|
| | 🔲 确定/大概率/不确定/存疑 | | 🔲 低/中/高 |
| | 🔲 确定/大概率/不确定/存疑 | | 🔲 低/中/高 |
| | 🔲 确定/大概率/不确定/存疑 | | 🔲 低/中/高 |

> ⚠️ 如果关键假设均为"存疑"且验证成本高，命题本身可能需要重新审视。

## 三、审视理由与证据

> 目标：检验支撑结论的逻辑链条是否坚实。

### 3.1 逻辑链条拆解

```
前提A + 前提B + 前提C
        ↓
    中间结论
        ↓
    最终结论
```

| 步骤 | 从 → 到 | 逻辑类型 | 是否必然成立 |
|------|---------|---------|------------|
| 1 | A → | 🔲 演绎/归纳/类比/权威 | 🔲 必然/大概率/可能/存疑 |
| 2 | → | 🔲 演绎/归纳/类比/权威 | 🔲 必然/大概率/可能/存疑 |
| 3 | → 结论 | 🔲 演绎/归纳/类比/权威 | 🔲 必然/大概率/可能/存疑 |

引导追问：
- 从A到B的推理必然成立吗？有没有A成立但B不成立的情况？
- 这里用的是归纳推理——样本够不够？有没有选择性偏差？
- 这个结论是否过度依赖某个权威说法？

### 3.2 反面证据

| 我知道的正面证据 | 可能的反面证据 | 我是否认真找过反面证据？ |
|----------------|---------------|----------------------|
| | | 🔲 是/否 |
| | | 🔲 是/否 |
| | | 🔲 是/否 |

引导追问：
- 有没有相反的证据或案例？
- 我是否主动寻找过反面证据，还是只看到支持自己的？
- 如果有人要反驳我，他会拿出什么？

## 四、探索替代观点

> 目标：跳出单一视角，看到其他可能性。

### 4.1 角色翻转

| 立场 | 他们会怎么说？ | 最有说服力的点 | 我如何回应？ |
|------|--------------|--------------|------------|
| 反对者 | | | |
| 中立旁观者 | | | |
| 受影响方（具体：___） | | | |
| 一线执行者 | | | |

引导追问：
- 与我观点相反的人会怎么论证？他们最有说服力的点是什么？
- 如果站在X的立场上，这个问题看起来是什么样？
- 一个完全不懂这个行业的人，会问什么"傻问题"？

### 4.2 中间立场与更复杂模型

| 选项 | 描述 | 兼顾了哪些合理之处？ |
|------|------|-------------------|
| 我的原始立场 | | |
| 对立立场 | | |
| **可能的中间方案** | | |
| **更复杂的模型** | | |

引导追问：
- 有没有一种中间立场，可能包含双方的合理之处？
- 这个问题是否不是二选一，而是一个光谱？
- 能否"既要又要"，代价是什么？

## 五、检验隐含后果与影响

> 目标：推演到极限，看逻辑结果是否可接受。

### 5.1 极端化推演

| 如果这个原则被普遍化…… | 结果 |
|----------------------|------|
| 如果所有人都按这个逻辑行事 | |
| 如果这个规则变成组织默认规则 | |
| 如果持续执行5年 | |

引导追问：
- 如果每个人都按你这个原则去做，世界/组织会变成什么样？
- 按照你的逻辑发展下去，会产生哪些你可能不喜欢的后果？
- 这个决定在1年后/3年后/5年后，分别会带来什么影响？

### 5.2 二阶效应

| 一阶结果（预期的） | 二阶效应（可能随之而来的） | 三阶效应 | 是否可接受？ |
|-------------------|------------------------|---------|------------|
| | | | 🔲 是/否 |
| | | | 🔲 是/否 |
| | | | 🔲 是/否 |

### 5.3 不可逆后果

| 后果 | 是否可逆 | 恢复成本 | 是否有退路 |
|------|---------|---------|-----------|
| | 🔲 可逆/不可逆 | | 🔲 有/无 |
| | 🔲 可逆/不可逆 | | 🔲 有/无 |

> ⚠️ 存在不可逆后果时，需要更充分的第一至三层论证。

## 六、反诘问题本身

> 目标：最高层次的反思——我们问的问题是不是个好问题？

### 6.1 问题重构

| 原始问题 | 可能的更根本问题 | 为什么后者更值得讨论？ |
|---------|----------------|---------------------|
| | | |
| | | |

引导追问：
- 我们是否应该先问"这件事真的值得做吗"，再问"怎么做"？
- 这个问题本身是否隐含了某个错误的预设？
- 我们是否忽略了比这个问题更根本的问题？
- 如果重新定义问题，答案会不会完全不同？

| 隐含预设 | 这个预设是否合理？ | 换一个预设会怎样？ |
|---------|------------------|------------------|
| | 🔲 合理/存疑/不合理 | |
| | 🔲 合理/存疑/不合理 | |

### 6.3 问题升级

| 层次 | 问题 |
|------|------|
| 现象层 | 我们在解决什么表面问题？ |

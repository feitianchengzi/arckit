**核心问题：** 画像是否足够具体？市场是否聚焦？

| 维度 | 填写 |
|------|------|
| 用户角色/身份 | |

| 是否已有典型用户样本 | 🔲 有（列举：） / 🔲 无 |

**聚焦度自评：** 🔲 非常具体 / 🔲 较清晰 / 🔲 还比较模糊

---

### 3.2 触发场景

**核心问题：** 场景频次如何？是刚需还是想要？

| 维度 | 填写 |
|------|------|

| 需求类型 | 🔲 不得不（刚需） / 🔲 想要（增益） / 🔲 偶尔想到（弱需求）|

**场景价值自评：** 🔲 高频刚需 / 🔲 高频非刚需 / 🔲 低频刚需 / 🔲 低频非刚需

### 3.5 核心解决方法

**核心问题：** 方案是真壁垒还是表层包装？是否有护城河？

| 维度 | 填写 |
|------|------|
| 核心方法描述（用一句话说清楚怎么解决） | |
| 与现有方案的本质差异 | |
| 壁垒类型 | 🔲 技术壁垒 / 🔲 数据壁垒 / 🔲 网络效应 / 🔲 品牌/信任 / 🔲 暂无明显壁垒 |
| 可被复制的难度 | 🔲 难（6个月+） / 🔲 中（3-6个月） / 🔲 易（随时可复制） |

**方案独特性自评：** 🔲 核心能力 / 🔲 有差异但可被替代 / 🔲 表层包装

### 3.6 预期价值

**核心问题：** 能量化吗？用户愿意为此付出代价吗？

| 价值类型 | 描述 | 量化估算 |
|----------|------|---------|
| 节省时间 | | 每次节省约 __ 分钟，每月约 __ 小时 |
| 降低成本 | | 降低约 __%，折合约 __ 元/月 |
| 提升收益 | | 预期增收约 __ 元/月 或提升 __% |
| 体验/情感价值 | | （描述，难以量化）|

**行为承诺（用户愿意为此做什么）：**
- 🔲 付费金额：___元/月
- 🔲 切换工具（放弃现有方案）
- 🔲 主动推荐给他人
- 🔲 参与内测 / 提供反馈

## SkillOps: arc-kit

Visibility: `public`

### Use in SkillOps Desktop

1. Open SkillOps.
2. Click **Add Skill project**.
3. Enter `github.com/feitianchengzi/arckit/tree/main/engineering` as the GitHub source.
4. Choose a profile and add an application target.

### Profiles

- `arc-kit/ios`: arckit-code-swiftui

### CLI

```bash
skillshare install github.com/feitianchengzi/arckit/tree/main/engineering --track --all && skillshare sync
npx skills add github.com/feitianchengzi/arckit/tree/main/engineering
```
